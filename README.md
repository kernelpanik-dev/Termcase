# Termcase
Termcase is a simple , command line interface program that allows you to open cs2 cases , inside your terminal!

// Features:
⟶ Knifes [Nomad,Butterfly,Karambit,Talon.. etc]
⟶ Floats [0.0 up to 0.85]
⟶ Roll counter [Shows your rolls]

! More features such as finishes , etc. will be added! :D
